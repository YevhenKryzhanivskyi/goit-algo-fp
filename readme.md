# Моделювання кидання двох кубиків методом Монте-Карло


## Реалізація
- Алгоритм симулює велику кількість кидків кубиків (`n_trials = 100_000`).
- Для кожного кидка обчислюється сума чисел на кубиках.
- Підраховується кількість появ кожної суми та визначається емпірична ймовірність.
- Аналітичні ймовірності задані відповідно до класичного розподілу (від 2 до 12).
- Результати виводяться у вигляді графіку.

## Висновки
1.1 Емпіричні ймовірності, отримані методом Монте-Карло, дуже близькі до аналітичних значень.  
1.2 Різниця між ними є незначною (порядку `10^-4`), що підтверджує правильність реалізації алгоритму.  
1.3 Симуляція методом Монте-Карло підтверджує теоретичні розрахунки розподілу сум при киданні двох кубиків.
